# 📚 Python Book Scraper – Books Data Extraction System

## 📌 Overview

The Python Book Scraper is a web scraping project that automates the extraction of book information from a public website. Instead of manually collecting book details, this application gathers data from multiple pages and stores it in a structured CSV file for further analysis and reporting.

This project demonstrates Python automation, web scraping, and data extraction techniques using the Requests and BeautifulSoup libraries.

---

## 🚀 Problem Statement

Collecting book information manually from websites is time-consuming, repetitive, and prone to human error. This project solves that problem by automating the entire data collection process, making it faster, more accurate, and suitable for analysis.

---

## ✨ Features

- Extracts book information from multiple pages
- Handles website pagination automatically
- Collects structured book data
- Exports data into CSV format
- Eliminates manual data collection
- Fast and efficient scraping process

---

## 📊 Data Extracted

The scraper collects information such as:

- Book Title
- Price
- Rating
- Availability Status
- Product Link
- Category (if available)

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/book-scraper.git
```

2. Navigate to the project folder

```bash
cd book-scraper
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the scraper

```bash
python scraper.py
```
